# Akash DTH IR Codes

This repository contains discrete infrared (IR) codes for the Akash DTH set-top box, specifically for the **N8784B** remote model. These codes can be imported into universal remote applications like [IRplus](https://play.google.com/store/apps/details?id=net.binarymode.android.irplus) to control your Akash DTH set-top box using your smartphone or other IR-enabled devices.

## How to Import IR Codes into IRplus App

Follow these steps to import the `Akash DTH.irplus` file into your IRplus application:

1. **Step 1: Open IRplus and navigate to the import option.**

   ![IRplus Import Step 1](/home/ubuntu/Akash_DTH/Akash/Image/import_step1.jpg)

2. **Step 2: Select the `Akash DTH.irplus` file from your device storage.**

   ![IRplus Import Step 2](/home/ubuntu/Akash_DTH/Akash/Image/import_step2.jpg)

Once imported, you can start controlling your Akash DTH set-top box directly from the IRplus app.
